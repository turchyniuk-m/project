# 📊 E-commerce Sales Analytics Project

## 🚀 Project Overview
This project presents an end-to-end analysis of e-commerce sales data using SQL, Python, and Tableau.  
It covers data extraction from BigQuery, data cleaning, analysis, and visualization to generate business insights.

---

## 🎯 Objectives
- Understand customer behavior and sales patterns  
- Analyze revenue, profit, and order distribution  
- Evaluate delivery performance and its impact on profit  
- Identify trends, seasonality, and growth opportunities  

---

## 🛠 Tools & Technologies
- Python (pandas, numpy, matplotlib, seaborn)
- SQL (Google BigQuery)
- Tableau
- Google Colab

## 📌 Key Insights
- Certain product categories generate significantly higher revenue  
- Delivery time impacts profitability and customer behavior  
- Some countries outperform others in terms of sales volume  
- Clear seasonal patterns are observed in weekly sales

## ▶️ How to Run the Project

### Option 1: Using BigQuery
1. Set up Google Cloud credentials  
2. Run the notebook using SQL query from `sql/main_query.sql`  

### Option 2: Using Sample Data
1. Open notebooks in Google Colab  
2. Load data from `data/sample_data.csv`
