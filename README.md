# 📊 E-commerce Sales Analytics Project

## 🚀 Project Overview
This project presents an end-to-end analysis of e-commerce sales data using SQL, Python, and Tableau.  
It covers data extraction from BigQuery, data cleaning, analysis, and visualization to generate business insights.

---

## 🎯 Objectives
- Understand customer behavior and sales patterns  
- Analyze revenue, profit, and order distribution  
- Evaluate delivery performance and its impact on profit  
- Identify trends, seasonality, and growth opportunities  

---

## 🛠 Tools & Technologies
- Python (pandas, numpy, matplotlib, seaborn)
- SQL (Google BigQuery)
- Tableau
- Google Colab
